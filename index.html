над которым я работал solo или в команде

Концепт проект для Rimowa

Система обработки заказов для известного производителя чемоданов Rimowa.

Проект основан на связке React, Material Ui, Node.js и PostrgreSQL. Программа предоставляет окно для ввода такой информации как серийный номер товара, имя и фамилия клиента, уникальный номер заказа итд. Данные отправляются на бэкэнд, где хранятся в базе данных. Специалист по чинке чемоданов имеет доступ к отдельной, защищенной паролем странице, где он он может просматривать заказы, помечать их как завершенные а так же скачивать список заказов как excel файл

Jazz Trainer Web App

Онлайн тренажер по джазовой гармонии.

Web-app по практике джазовой гармонии. Задается темп метронома, составляетя массив из нот, добавляются элементы муз. аккордов. Под метроном. заданная комбинация аккордов сменяется через промежуток времени, позволяя игроку практиковать аккорды на своем инструменте.

Персональный сайт soltonbaev.com

Мой личный сайт основанный на быстром движке Grav.

В основу сайта лежит flat file плафторма Grav, позволяющая загружать markdown файлы и динамически конвертировать их в HTML страницы. Таким образом обеспечивается быстрота работы сайта и избежания сложности с редактированием HTML кода.
